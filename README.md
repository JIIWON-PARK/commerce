3. 경제활동인구(2024)

[주요 전처리 요약]
1. 경제활동인구 파일과 상권분석 파일의 행정구역을 동일하게 만들기 위해 기존 경제활동인구의 서울 동대문구 -> 동대문구로 변경
2. 경제활동인구의 기간과 상권분석 파일의 기간이 반기별, 월별로 다르기 때문에 분기 기간 테이블 생성 후 기준 동일하게 맞춤

[테이블 별 처리방식]
 1. 상권매출
    원래의 형태 - 자치구/업종/분기별 매출
    전처리 - 기준_년분기_코드에서 기준년분기코드
 2. 점포현황
    원래의 형태 - 자치구/업종/분기별 점포수
    전처리 - 상권매출과 같은 분기 코드로 사용
 3. 경제활동인구
    원래의 형태 - 자치구(서울 00구)/성별/상반기/하반기
    전처리 - 상반기/하반기 세로형 변환
4. 기간(새로만듦)
   전처리 - 분기와 반기 연결 기준으로 사용

[경제활동 1인당 매출 계산식]
1. 상권매출에서 자치구별 총매출 계산
2. 성별 합산해 자치구별 상반기/하반기 경제활동인구 계산
3. 상반기/하반기 평균으로 연평균 경제활동인구 계산
4. 총매출 ÷ 연평균 경제활동인구 수식 사용

[종합 인사이트]
2024년 기준 서울시 상권은 전반적으로 주중 매출 비중이 높아 평일 중심의 소비 구조를 보임 그러나 자치구별로 주말 매출 비중과 객단가, 경제활동인구 대비 매출 수준은 다르게 나타나 지역별 상권 성격에 차이 있음
강남구는 총매출 규모가 큰 대표 상권으로 볼 수 있지만, 객단가 측면에서는 용산구가 높게 나타났고, 경제활동인구 1명당 매출은 중구와 종로구가 높게 나타남
폐업률은 일부 오락·여가·특수 업종에서 높게 나타나, 매출 규모뿐 아니라 업종별 안정성까지 함께 고려해야함
2024년 기준이므로 현재와 동일하지 않을 수 있음
